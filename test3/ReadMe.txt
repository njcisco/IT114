* To Run the Client UI
    1. Open a terminal
    2. Compile using 'javac Main.java'
    3. Run using 'java Main'

* To run the Server
    1. Open a separate terminal
    2. Compile using 'javac SampleSocketServer.java'
    3. Run using 'java SampleSocketServer <Port Number>'
